# AI Assistant Java Prototype – Project Structure

This repository contains a prototype AI voice assistant built with Java.

## Main Components

src/main/java
Main Java source code.

devlog
Development notes and experiments.

target
Compiled output files (ignored by Git).

pom.xml
Maven build configuration.

README.md
Project overview and features.

## Features

• Voice recording from microphone  
• Speech-to-text processing  
• AI response generation  
• Chat-style UI display  
• HTTP communication with AI APIs

## Future Improvements

• Streaming voice recognition  
• Plugin architecture  
• Better chat interface  
• Model switching support
