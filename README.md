\# AI Voice Assistant (Java)



A simple AI-powered voice assistant built in Java.  

The application records user audio, converts speech to text, sends the request to an AI model, and displays the response.



This project demonstrates how to integrate voice input, HTTP APIs, and AI services in a Java desktop application.



\---



\## 🚀 Features



\- 🎤 Voice recording from microphone

\- 🧠 AI response generation using OpenAI API

\- 💬 Chat-style response display

\- ⚡ Lightweight Java desktop application

\- 🌐 HTTP communication with external APIs



\---



\## 🧱 Project Architecture



User Voice  

↓  

Audio Recording (Java)  

↓  

Speech → Text  

↓  

AI Request (OpenAI API)  

↓  

AI Response  

↓  

Display in Chat Window
